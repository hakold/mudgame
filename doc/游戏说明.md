# 游戏说明

## 一、游戏介绍

侠客行是一款文字RPG游戏，以武侠小说为背景，采用MUD（Multi-User Dungeon）玩法。玩家通过文字命令与游戏世界交互，体验武侠江湖的精彩故事。

## 二、基本操作

### 2.1 注册与登录

1. 打开游戏网页
2. 点击"注册"创建账号
3. 填写用户名、密码、邮箱、角色名、性别
4. 注册成功后自动登录进入游戏

### 2.2 命令系统

游戏主要通过命令进行操作。在命令输入框输入命令，按回车或点击"发送"执行。

**常用命令:**

| 命令 | 说明 | 示例 |
|------|------|------|
| help | 查看帮助 | help |
| look / l | 查看当前房间 | look |
| go / move | 移动到指定方向 | go north |
| n / s / e / w | 快捷移动（北/南/东/西） | n |
| kill / attack | 攻击怪物 | kill monster_wild_boar |
| skill | 学习技能 | skill shaolin_changquan |
| use | 使用物品 | use item_hp_potion_small |
| quest | 任务操作 | quest accept quest_clear_forest |
| chat | 世界聊天 | chat 大家好 |
| say | 房间聊天 | say 有人吗 |
| tell | 私聊 | tell 张三 你好 |
| faction | 门派操作 | faction join shaolin |

## 三、角色系统

### 3.1 属性说明

| 属性 | 说明 | 影响 |
|------|------|------|
| 力量 | 物理力量 | 攻击力 |
| 敏捷 | 动作敏捷 | 闪避、暴击 |

### 3.2 升级系统

- 通过战斗、完成任务获得经验
- 经验达到要求后自动升级
- 每升一级获得3点自由属性点
- 可在角色面板分配属性点

- **竞技场**: PVP战斗

## 五、战斗系统

### 5.1 战斗流程

1. 在房间中点击怪物名称或输入 `kill <怪物ID>`
2. 进入战斗界面
3. 选择行动：攻击、防御、使用技能、逃跑
4. 战斗结束后获得经验和金币

### 5.2 战斗属性

- **HP**: 生命值，归零则死亡
- **MP**: 内力值，使用技能消耗
- **攻击力**: 造成伤害的基础
- **防御力**: 减少受到的伤害
- **闪避**: 躲避攻击的概率

### 5.3 战斗技巧

- 注意HP/MP状态，及时使用药品
- 合理使用技能，注意MP消耗
- 面对强敌可以先防御或逃跑
- 装备好的武器和护甲提升战斗力

- 装备有等级要求
- 装备可以强化提升属性

## 八、任务系统

### 8.1 任务类型

| 类型 | 说明 |
|------|------|
| 杀怪任务 | 击杀指定数量的怪物 |
| 收集任务 | 收集指定物品 |
| 探索任务 | 到达指定地点 |
| 门派任务 | 加入门派相关 |

### 8.2 任务流程

1. 与NPC对话，接取任务
2. 完成任务目标
3. 返回NPC处完成任务
4. 领取奖励（经验、金币、物品）

### 8.3 任务奖励

- 经验值：提升等级
- 金币：购买物品
- 物品：装备、消耗品等

## 十二、新手建议

1. 先在新手村熟悉操作
2. 完成村长的任务获取奖励
3. 购买基础装备和药品
4. 到森林练功提升等级
5. 加入门派学习技能
6. 挑战更强的怪物和任务
