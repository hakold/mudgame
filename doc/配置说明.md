# 配置说明

本文档说明如何通过配置文件修改游戏内容，无需修改代码。

## 一、配置文件位置

所有配置文件位于 `Gameproject/config/` 目录：

```
config/
├── excel/          # Excel源文件（可编辑）
│   ├── maps.xlsx
│   ├── rooms.xlsx
│   ├── npcs.xlsx
│   ├── monsters.xlsx
│   ├── items.xlsx
│   ├── skills.xlsx
│   ├── quests.xlsx
│   └── factions.xlsx
│
└── json/           # JSON配置文件（游戏读取）
    ├── maps.json
    ├── rooms.json
    ├── npcs.json
    ├── monsters.json
    ├── items.json
    ├── skills.json
    ├── quests.json
    └── factions.json
```

## 二、配置流程

### 方式一：直接编辑JSON（推荐新手）

直接编辑 `config/json/` 目录下的 `.json` 文件，修改后重启服务器生效。

### 方式二：Excel编辑后转换（推荐批量修改）

1. 编辑 `config/excel/` 目录下的 `.xlsx` 文件
2. 运行转换脚本生成JSON
3. 重启服务器

```bash
cd ~/Desktop/Gameproject/server
npm run import-config
```

## 三、各配置文件说明

### 3.1 maps.json - 地图配置

定义游戏中的大区域（地图）。

```json
[
  {
    "id": "village",              // 地图唯一ID
    "name": "新手村",             // 显示名称
    "description": "描述文字",    // 地图描述
    "level": "1-10",              // 推荐等级范围
    "rooms": ["village_center"]   // 包含的房间ID列表
  }
]
```

**添加新地图步骤:**
1. 在 maps.json 中添加新地图配置
2. 在 rooms.json 中添加该地图的房间
3. 重启服务器

### 3.2 rooms.json - 房间配置

定义具体的房间，玩家移动的最小单位。

```json
[
  {
    "id": "village_center",       // 房间唯一ID
    "name": "村庄广场",           // 显示名称
    "description": "详细描述",    // 房间描述
    "mapId": "village",           // 所属地图ID
    "exits": [                    // 出口列表
      {
        "direction": "north",     // 方向名称
        "roomId": "village_inn"   // 目标房间ID
      }
    ],
    "features": ["石碑"],         // 房间特征
    "services": ["rest"]          // 可用服务
  }
]
```

**方向说明:**
- 基本方向: north, south, east, west
- 特殊方向: up, down, in, out
- 自定义方向: 如 "temple", "stairs"

**添加新房间步骤:**
1. 在 rooms.json 中添加房间配置
2. 确保出口指向的房间ID存在
3. 重启服务器

### 3.3 npcs.json - NPC配置

定义非玩家角色。

```json
[
  {
    "id": "npc_village_chief",    // NPC唯一ID
    "name": "村长",               // 显示名称
    "description": "描述",        // NPC描述
    "roomId": "village_center",   // 所在房间ID
    "type": "quest_giver",        // NPC类型
    "dialogues": {                // 对话内容
      "greeting": "欢迎！",
      "quest_available": "有任务"
    },
    "quests": ["quest_001"],      // 可接取的任务
    "items": ["item_001"],        // 商店出售的物品
    "skills": ["skill_001"]       // 可教授的技能
  }
]
```

**NPC类型:**
- `quest_giver`: 任务NPC
- `shop`: 商店NPC
- `trainer`: 技能导师
- `faction`: 门派NPC
- `service`: 服务NPC（客栈、铁匠等）

### 3.4 monsters.json - 怪物配置

定义可战斗的怪物。

```json
[
  {
    "id": "monster_wild_boar",    // 怪物唯一ID
    "name": "野猪",               // 显示名称
    "description": "描述",        // 怪物描述
    "level": 3,                   // 怪物等级
    "hp": 80,                     // 最大HP
    "attack": 8,                  // 攻击力
    "defense": 5,                 // 防御力
    "exp": 15,                    // 击杀经验
    "gold": 5,                    // 击杀金币
    "roomId": "forest_entrance",  // 所在房间
    "drops": [                    // 掉落物品
      {
        "itemId": "item_boar_meat",
        "quantity": 1,
        "chance": 0.3             // 掉落概率
      }
    ],
    "dropRate": 0.5,              // 总掉落率
    "isBoss": false               // 是否为BOSS
  }
]
```

**数值设计建议:**
- HP = 等级 × 20 ~ 30
- 攻击 = 等级 × 2 ~ 3
- 防御 = 等级 × 1 ~ 2
- 经验 = 等级 × 5 ~ 10
- 金币 = 等级 × 2 ~ 5

### 3.5 items.json - 物品配置

定义所有物品。

```json
[
  {
    "id": "weapon_wooden_sword",  // 物品唯一ID
    "name": "木剑",               // 显示名称
    "description": "描述",        // 物品描述
    "type": "equipment",          // 物品类型
    "slot": "weapon",             // 装备槽位
    "rarity": "common",           // 稀有度
    "attributes": {               // 属性加成
      "attack": 3
    },
    "requireLevel": 1,            // 等级要求
    "price": 20,                  // 购买价格
    "sellPrice": 5                // 出售价格
  }
]
```

**物品类型:**
- `equipment`: 装备（可穿戴）
- `consumable`: 消耗品（可使用）
- `material`: 材料（可出售）

**装备槽位:**
- `weapon`: 武器
- `armor`: 护甲
- `helmet`: 头盔
- `boots`: 鞋子
- `accessory`: 饰品
- `ring`: 戒指

**稀有度:**
- `common`: 普通（白色）
- `uncommon`: 优秀（绿色）
- `rare`: 稀有（蓝色）
- `epic`: 史诗（紫色）
- `legendary`: 传说（金色）

**消耗品配置:**
```json
{
  "id": "item_hp_potion_small",
  "name": "小还丹",
  "type": "consumable",
  "effects": [
    { "type": "heal_hp", "value": 50 }
  ],
  "stackable": true,
  "maxStack": 99
}
```

### 3.6 skills.json - 技能配置

定义所有技能。

```json
[
  {
    "id": "shaolin_changquan",    // 技能唯一ID
    "name": "少林长拳",           // 显示名称
    "description": "描述",        // 技能描述
    "type": "attack",             // 技能类型
    "damageAttribute": "strength",// 伤害属性
    "baseDamage": 15,             // 基础伤害
    "damageGrowth": 5,            // 每级伤害成长
    "attributeRatio": 1.0,        // 属性系数
    "baseCost": 5,                // 基础MP消耗
    "costGrowth": 2,              // 每级消耗成长
    "baseCooldown": 3,            // 基础冷却（秒）
    "cooldownReduce": 0.2,        // 每级冷却减少
    "minCooldown": 1,             // 最小冷却
    "maxLevel": 10,               // 最高等级
    "expBase": 100,               // 升级经验基数
    "expExponent": 1.5,           // 升级经验指数
    "learnCost": 50,              // 学习金币
    "requireLevel": 5,            // 学习等级
    "factionRequired": "shaolin"  // 门派限制
  }
]
```

**技能类型:**
- `attack`: 攻击技能
- `heal`: 治疗技能
- `buff`: 增益技能
- `special`: 特殊技能

**伤害公式:**
```
伤害 = baseDamage + level × damageGrowth + 属性值 × attributeRatio
```

### 3.7 quests.json - 任务配置

定义所有任务。

```json
[
  {
    "id": "quest_clear_forest",   // 任务唯一ID
    "name": "清理森林",           // 显示名称
    "description": "描述",        // 任务描述
    "type": "kill",               // 任务类型
    "npcId": "npc_village_chief", // 发布NPC
    "objectives": [               // 任务目标
      {
        "id": "kill_boar",
        "type": "kill",
        "target": "monster_wild_boar",
        "targetCount": 3,
        "description": "击杀3只野猪"
      }
    ],
    "prerequisites": [],          // 前置任务
    "requireLevel": 1,            // 等级要求
    "reward": {                   // 奖励
      "exp": 50,
      "gold": 30,
      "items": ["item_hp_potion_small"]
    },
    "repeatable": false           // 是否可重复
  }
]
```

**任务类型:**
- `kill`: 杀怪任务
- `collect`: 收集任务
- `explore`: 探索任务
- `faction`: 门派任务
- `pvp`: PVP任务

### 3.8 factions.json - 门派配置

定义所有门派。

```json
[
  {
    "id": "shaolin",              // 门派唯一ID
    "name": "少林派",             // 显示名称
    "description": "描述",        // 门派描述
    "leader": "方证大师",         // 门派掌门
    "location": "mountain_temple",// 门派驻地
    "requireLevel": 5,            // 加入等级
    "bonus": {                    // 属性加成
      "strength": 2,
      "constitution": 2
    },
    "skills": ["shaolin_changquan"],// 门派技能
    "color": "#FFD700"            // 显示颜色
  }
]
```

## 四、数值设计参考

### 4.1 等级经验公式

```
升级所需经验 = expBase × level^expExponent
```

推荐参数:
- expBase = 100
- expExponent = 1.5

示例:
- 1级升2级: 100 × 1^1.5 = 100
- 10级升11级: 100 × 10^1.5 = 1000
- 50级升51级: 100 × 50^1.5 = 7071

### 4.2 HP/MP公式

```
最大HP = 100 + 体质 × 10 + 等级 × 5
最大MP = 50 + 悟性 × 5 + 等级 × 2
```

### 4.3 战斗公式

```
攻击力 = 力量 × 2 + 等级
防御力 = 体质 + 等级
闪避率 = 敏捷 / 100

伤害 = 攻击力 - 防御力（最小为1）
```

### 4.4 技能伤害公式

```
技能伤害 = baseDamage + level × damageGrowth + 属性 × attributeRatio
```

## 五、配置后台

游戏提供Web配置后台，可在浏览器中直接修改配置。

### 5.1 访问配置后台

1. 登录管理员账号（admin）
2. 点击"管理后台"
3. 选择"配置管理"

### 5.2 功能说明

- **地图管理**: 添加/编辑/删除地图和房间
- **NPC管理**: 添加/编辑/删除NPC
- **怪物管理**: 添加/编辑/删除怪物
- **物品管理**: 添加/编辑/删除物品
- **技能管理**: 添加/编辑/删除技能
- **任务管理**: 添加/编辑/删除任务
- **门派管理**: 添加/编辑/删除门派

### 5.3 注意事项

- 修改配置后需要重启服务器生效
- 删除配置前确保没有引用关系
- 建议先在测试环境验证配置

## 六、常见配置场景

### 场景1: 添加新区域

1. 在 maps.json 添加新地图
2. 在 rooms.json 添加多个房间
3. 在 npcs.json 添加区域NPC
4. 在 monsters.json 添加区域怪物
5. 重启服务器

### 场景2: 添加新门派

1. 在 factions.json 添加门派配置
2. 在 skills.json 添加门派技能
3. 在 npcs.json 添加门派NPC
4. 重启服务器

### 场景3: 添加新任务

1. 在 quests.json 添加任务配置
2. 确保任务目标（怪物/物品）存在
3. 在对应NPC的quests列表中添加任务ID
4. 重启服务器

### 场景4: 调整数值平衡

1. 修改 monsters.json 中的怪物数值
2. 修改 skills.json 中的技能数值
3. 修改 items.json 中的物品属性
4. 重启服务器
5. 测试战斗效果
