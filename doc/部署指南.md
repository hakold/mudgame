# 部署指南

本文档面向不懂程序的用户，详细介绍如何搭建环境并运行游戏。

## 一、环境准备

### 1.1 安装 Node.js

Node.js 是运行游戏服务器的必需软件。

**Windows系统:**
1. 访问 https://nodejs.org/
2. 下载 LTS（长期支持版）安装包
3. 双击安装包，一路点击"下一步"完成安装
4. 打开"命令提示符"（按 Win+R，输入 cmd，回车）
5. 输入 `node -v`，如果显示版本号（如 v20.x.x），说明安装成功

**Mac系统:**
1. 访问 https://nodejs.org/
2. 下载 LTS 版本的 macOS 安装包
3. 双击安装包完成安装
4. 打开"终端"（在"应用程序"->"实用工具"中）
5. 输入 `node -v`，如果显示版本号，说明安装成功

### 1.2 安装 MongoDB

MongoDB 是存储游戏数据的数据库。

**Windows系统:**
1. 访问 https://www.mongodb.com/try/download/community
2. 选择 Windows 版本，下载 MSI 安装包
3. 双击安装包完成安装
4. MongoDB 会自动作为 Windows 服务启动

**Mac系统:**
1. 打开终端
2. 输入: `brew tap mongodb/brew`
3. 输入: `brew install mongodb-community`
4. 输入: `brew services start mongodb-community` 启动服务

**验证安装:**
- 打开终端/命令提示符
- 输入 `mongosh` 或 `mongo`
- 如果进入 MongoDB shell，说明安装成功

### 1.3 安装 Redis（可选）

Redis 用于缓存，提升性能。如果只是本地测试，可以跳过。

**Windows系统:**
1. 访问 https://github.com/microsoftarchive/redis/releases
2. 下载最新的 Redis-x64-*.msi
3. 安装后会自动启动服务

**Mac系统:**
1. 打开终端
2. 输入: `brew install redis`
3. 输入: `brew services start redis`

## 二、配置数据库连接

### 2.1 修改后端配置

1. 打开文件夹 `Gameproject/server/`
2. 找到文件 `.env`（如果没有，复制 `.env.example` 并重命名为 `.env`）
3. 用记事本或其他文本编辑器打开
4. 修改以下配置：

```
# MongoDB配置（本地安装）
MONGODB_HOST=localhost
MONGODB_PORT=27017
MONGODB_DATABASE=wuxia_mud

# Redis配置（如果安装了）
REDIS_HOST=localhost
REDIS_PORT=6379
```

**如果 MongoDB 和 Redis 在其他服务器:**
```
MONGODB_HOST=192.168.31.148
MONGODB_PORT=10055

REDIS_HOST=192.168.31.148
REDIS_PORT=16379
```

## 四、初始化数据库

首次运行需要初始化数据库，创建管理员账号：

```bash

```

看到以下输出说明成功：

```
[Init] 用户名: admin, 密码: admin123
```

## 七、常见问题

### Q: 启动后端报错 "MongoNetworkError"
A: MongoDB 没有启动或连接配置错误
- 检查 MongoDB 服务是否启动
- 检查 `.env` 中的连接配置是否正确

### Q: 启动后端报错 "Redis connection error"
A: Redis 没有启动，可以：
- 启动 Redis 服务
- 或者暂时不使用 Redis（需要修改代码）

### Q: 前端无法连接后端
A: 检查：
- 后端是否已启动
- 浏览器控制台是否有错误信息
- 防火墙是否阻止了连接

### Q: 登录失败
A: 检查：
- 数据库是否已初始化
- 用户名密码是否正确
- 后端日志是否有错误信息

## 八、生产环境部署

### 8.1 构建前端
```bash
cd ~/Desktop/Gameproject/client
npm run build
```
生成的文件在 `client/dist/` 目录

### 8.2 使用 PM2 管理后端
```bash
npm install -g pm2
cd ~/Desktop/Gameproject/server
pm2 start src/app.js --name wuxia-mud
pm2 save
pm2 startup
```

### 8.3 Nginx 配置示例
```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    # 前端静态文件
    location / {
        root /path/to/Gameproject/client/dist;
        try_files $uri $uri/ /index.html;
    }
    
    # API 代理
    location /api {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
    
    # WebSocket 代理
    location /socket.io {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
    }
}
```
