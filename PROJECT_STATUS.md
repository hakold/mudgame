## 门派系统

每个门派包含：
- 入门考核（独立任务，完成后方可加入）
- 四阶晋升：弟子→执事→长老→掌门（各有试炼任务）
- 贡献兑换系统（用门派贡献换功法学习）
- 专属NPC：招募者/守门/传功长老/兑换长老

## 功法书系统

- 56本功法书，分为初心者(60-80%)/中级(40-55%)/高级(25-35%)/绝学(10-20%)
- 5本稀有真本（可无视门派要求学习）
- 使用方式：`use <功法书名称>`

## 装备系统

全部位覆盖：武器(39) / 身体(18) / 头盔(13) / 靴子(13) / 戒指(8) / 项链(7)
等级覆盖：L1-L80全等级段均有对应装备

## 快速启动

1. 启动MongoDB和Redis
2. `cd server && npm run init-db`
3. `npm start` (后端) + `cd client && npm run dev` (前端)
4. 访问 http://localhost:5173
