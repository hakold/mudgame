### 1.2 副本系统架构背景

北大侠客行MUD采用**分布式副本机制**，在主流MUD中属首创。核心理念：
- **单线程瓶颈**: fluffos驱动只能在单核运行，玩家增加导致CPU 100%
- **多核利用**: 在主站外启动独立driver跑副本，利用OS多核调度
- **无缝切换**: 玩家进入副本仅需~2秒等待，体验与进入前一致
- **负载分担**: 副本上线后CPU从100%降至~50%，彻底消除性能焦虑

**副本通用规则**:
- 进入: 自动提示"正在进入副本"，buff/特技需重新激活
- 离开: `leave <自己的id>` 回到主站
- 物品: 携带进出，但顺序/状态可能变化；金钱出副本后恢复
- 区域: 仅限任务相关区域活动，超出自动移至入口
- 奖励: 一般在副本内领取

## 三、公共副本任务详细

### 3.1 藏经阁 (Cangjingge) — 潜行闯关型副本

| 维度 | 设计参数 |
|------|----------|
| **任务类型** | 潜行+迷宫探索+战斗 |
| **前置条件** | exp>15M，需易容成护寺僧兵 |
| **模式** | 副本模式、假死、单人/组队(上限6人)；单人有困难模式 |
| **次数** | 每天2次(0点重置) |
| **CD** | 无明确CD，有20分钟超时上限 |
| **奖励机制** | 积分制(三部分): ①杀怪积分(小怪1分/长老5分,上限100); ②固定通关积分(与人数正比); ③时间奖励(5分钟满分,每分钟-10分) |
| **掉落** | 50%概率掉落丹药(回气药/解毒丸/秘制金疮药/秘制养精丹)、极小概率掉易容包 |
| **兑换商店** | 三层藏经阁(2F需5万积分,3F需15万积分): 武功秘籍/知识技能/特效套装/丹药/宝石/经验丹/阅览证(1000分换4小时) |
| **核心玩法** | ①9×6矩阵迷宫(避开巡逻僧兵,看破次数有限)；②20房间随机迷宫(丢铜板标记,陷阱); ③遭遇战(白天敌人数量×2,精英长老会少林三绝技) |
| **难度梯度** | 白天>晚上(敌人数量2倍+伤害增加); 多人>单人; 困难模式(NPC经验多30%,奖励积分+80,时限8分钟) |
| **失败机制** | 看破次数耗尽→戒律院大牢; 超时20分钟→戒律院; 队伍死人→仅1/3积分 |

**设计要点**:
- 时间与难度挂钩:白天遭遇战数量×3几率、×2人数，鼓励夜间挑战
- 双迷宫差异化:第一层矩阵式(可look map)、第二层随机式(需铜板标记)
- 组队风险收益平衡:人数增加难度和奖励,死人就大幅扣分

### 3.2 万安塔 (Wan'anta) — 爬塔挑战型副本

| 维度 | 设计参数 |
|------|----------|
| **任务类型** | 爬塔+逐层战斗 |
| **前置条件** | 等级≥15 |
| **模式** | 副本模式、假死、单人/组队(经验差≤10%,组队NPC数量翻倍) |
| **CD** | 基础15分钟,每上一层-2分钟; 死亡重置15分钟; 6-7层CD单独计算; 中途离开需验证(输错23分钟/不输18分钟/正确-1分钟) |
| **层级** | 7层,武士数量: L1-3=1个, L4≤2, L5≤5, L6≤7, L7≤9 |
| **奖励** | 经验阶梯(L1-5: 2K-8K浮动, L6: ~40K, L7: ~110K); L7宝盒(地-天宝石+几率奖励一级武功); 掉落职业技能书/职业装备/五行禽兽宝石/玉石 |
| **核心玩法** | 每层杀完武士→up上楼; 用`jiangli`查看当前奖励→决策继续或`qiao luo`退出; 战斗后敲锣离开(有图片验证) |
| **难度梯度** | L1-4: 武士经验低于玩家; L5-6: 相当; L7: 高于玩家; 塔7次数≥300(困难)/≥500(地狱,前缀NPC) |
| **特殊奖励** | 塔7成功2000次→玄冥神掌(初始100级); 1500次以上极小概率获得; 后续每次爬塔问鹿杖客可提升玄冥神掌等级 |
| **失败惩罚** | 假死但扣除当前可获等量经验/潜能/声望; 身上物品全丢 |

**设计要点**:
- "电视答题"式心理博弈:拿当前奖励走人 vs 冒险冲更高层
- CD递减机制:爬越高CD越短,鼓励高手高效刷塔
- 长期目标:2000次塔7→玄冥神掌,形成超长线追求
- 稀有刷新:每层有概率只刷1个武士→赌狗机制

|------|----------|
| **任务类型** | 定时多人协作防御 |
| **前置条件** | 等级≥11 |
| **模式** | 假死副本模式、多人协作 |
| **时间** | 每日9:00/15:00/20:30/21:00四个时段 |
| **核心玩法** | ①修防御工事(dig,每次BUSY较长); ②运礌石滚木; ③四城门+帅府防御; ④杀蒙古兵和骑兵 |
| **胜负条件** | 帅府大厅沦陷→失败,奖励作废; 全灭蒙古兵→成功 |
| **奖励机制** | 按排名档位(1-5/6-10/11-20/21+),每档-25%奖励; 每破一门-25%; 杀敌<6个→仅20%奖励; 乘积关系 |
| **积分** | 藏经阁积分 |
| **后续活动** | 保卫结束后3分钟→蒙古奸细出现在荆州/汉口杀军民; 阻止≥30%→全体参与者1小时内主流任务≥100%; 失败→大宋1小时主流任务≤85% |

**设计要点**:
- 多层次惩罚乘积:排名×破门×杀敌数,促使全方面参与
- 定时事件创造玩家聚集效应（每日4场）
- 后续活动形成连锁效应（影响全服1小时任务效率）

### 3.4 鄱阳湖漂流 (Piaoliu) — 航海寻宝型副本

| 维度 | 设计参数 |
|------|----------|

| **前置条件** | 无明确限制 |

**设计要点**:
- 航行操控深度:至少10种船控指令,创造"驾船"沉浸感
- 属性轮换:每小时换挂钩属性,鼓励多角色/多时段
- 家信机制:漂流产出"信"→交主流任务NPC→下一次任务奖励翻倍(跨系统激励)

### 3.7 陆乘风破阵 (Pozhen) — 解谜型副本

| 维度 | 设计参数 |
|------|----------|
| **任务类型** | 八卦阵解谜+战斗 |
| **前置条件** | 无明确限制(建议占卜81级) |
| **模式** | 副本模式、假死(走阵死真死/被恶人打死假死) |
| **核心玩法** | ask lu about pozhen→到地点zhanbu -maze→八卦阵(8方向)→闪烁提示→试错排除→走对3次→进入阵眼→打死恶人→回原点交任务 |
| **占卜作用** | 等级高→闪烁方向多→容易排除; 桃花岛可学到301级/别派201级 |
| **限制** | 阵中不能yun/服药; 走错扣血→血量低自动出阵失败 |
| **奖励系数** | 别派:<1.5M=100%, <10M=80%, ≥10M=70%; 桃花:<10M=100%, ≥10M=110%; 桃花额外门忠+1; 图片加倍/hard模式150% |
| **奖励幅度** | 介于慕容和韩员外之间 |
| **关联技能** | 与先天福缘/悟性/占卜术等级正相关 |

**设计要点**:
- 排除法解谜:闪烁提示+试错=推理玩法
- 门派差异化:桃花岛有优势(更高占卜上限+110%奖励+门忠)
- 无法恢复机制:走阵消耗不可逆,战斗时残血增加紧张感
- CD在1.5M以下奖励100%→鼓励新手期做

### 5.2 奖励机制设计原则

1. **分层激励**: 不同经验段玩家适用不同任务 → 持续成长感
2. **排名竞争**: 保卫襄阳按排名分档 → PvE中的PvP竞争
3. **长期目标**: 塔7×2000次/剿匪×100次 → 超长线留存
4. **跨系统联动**: 漂流家信→主流任务奖励翻倍; 保卫→全服任务效率
5. **积分经济**: 藏经阁积分形成独立的通货体系
6. **风险收益**: 高难度=真死+高奖励; 低难度=假死+低奖励

## 六、副本设计要点总结

### 6.1 副本通用模式

| 特性 | 说明 |
|------|------|
| 死亡处理 | 大部分假死(藏经阁/万安塔/保卫/漂流/韩世忠/破阵),高难剿匪真死 |
| 物品安全 | 死亡后尸体无法捡回(塔/韩世忠等),建议贵重物品存放 |
| 组队支持 | 藏经阁(6人)/万安塔(经验差≤10%)/漂流(2-4人)/剿匪(大规模) |
| CD机制 | 分层CD:成功CD短/失败CD长/高层CD短; 漂流有每日上限 |
| 验证码 | 万安塔敲锣/漂流传音 出图片验证,防机器人 |

### 6.2 核心设计模式

1. **迷宫探索**: 藏经阁(矩阵+随机)/韩世忠(方位推理)/破阵(八卦排除)
2. **爬塔清层**: 万安塔(逐层递增+赌博决策)
3. **定时防御**: 保卫襄阳(多人协作+排名竞争+后续影响)
4. **航海模拟**: 鄱阳湖(船控系统+属性挂钩)
5. **大型Raid**: 剿匪(多路线+多支线+宝库分配)
6. **辨识推理**: 破解武功(武功描述匹配)

### 6.3 难度梯度设计

| 维度 | 实现方式 |
|------|----------|
| 玩家等级 | 任务有最低等级/经验要求 |
| 昼夜影响 | 藏经阁白天敌人×2; 保卫固定时段 |
| 动态难度 | 万安塔次数≥300/500难度跃升; 藏经阁困难模式 |
| 人数弹性 | 组队人多→NPC数量/强度增加 |
| 自行选择 | 漂流三种模式; 剿匪四种难度; 破阵hard模式 |
| 时间压力 | 藏经阁5分钟满分→8分钟递减; 保卫定时开始 |
