## 1. 属性体系

### 1.1 先天属性（角色创建时确定）

PKUXKX MUD 中的先天属性系统，是角色的根本特质，创建时随机分配，游戏中极少能改变：

| 属性 | 英文名 | 含义 | 游戏影响 |
|------|--------|------|----------|
| **臂力** | strength | 力量/体格 | 影响攻击伤害、负重上限 |
| **悟性** | intelligence | 学习能力 | 影响读书效率、技能学习速度、钻研上限 |
| **根骨** | constitution | 体质根基 | 影响 HP 上限、恢复速度、内力上限 |
| **身法** | dexterity | 敏捷/灵活 | 影响闪避、攻击速度、轻功效果 |
| **容貌** | charisma | 外貌 | 影响 NPC 交互、部分任务 |
| **福缘** | luck | 运气 | 影响掉落率、随机事件 |

### 1.2 本项目的属性设计

当前实现采用 5 属性体系（简化自 MUD 的 6 属性）：

| 属性 | 初始值 | 计算公式 | 说明 |
|------|--------|----------|------|
| **力量** (strength) | 10 | 攻击力 = 力量×2 + 等级 | 核心战斗属性 |
| **敏捷** (dexterity) | 10 | 闪避 = 敏捷/2 | 影响闪避与身法 |
| **体质** (constitution) | 10 | 防御力 = 体质 + 等级, HP = 100 + 体质×10 + 等级×5 | 生存属性 |
| **悟性** (intelligence) | 10 | MP = 50 + 悟性×5 + 等级×2 | 法术/内力属性 |
| **根骨** (charisma) | 10 | (当前无战斗公式) | 保留用作交互属性 |

### 1.3 属性成长

| 途径 | 获得方式 | 说明 |
|------|----------|------|
| 升级加点 | 每级 +3 自由属性点 | 可自由分配至 5 项属性 |
| 门派进阶 | 进阶时 +5 自由属性点 | 弟子→执事→长老→掌门 |
| 装备加成 | 武器/防具提供 attack/defense/dexterity 加成 | 通过 Inventory.calculateAttributes() |

### 1.4 战斗属性汇总

```
最大HP   = 100 + 体质×10 + 等级×5
最大MP   = 50  + 悟性×5  + 等级×2
攻击力   = 力量×2 + 等级 + 装备attack
防御力   = 体质   + 等级 + 装备defense
闪避值   = 敏捷/2 + 装备dexterity/2
```

## 2. 战斗公式

### 2.1 核心伤害公式

```
baseDamage = max(1, attackerAtk - defenderDef × 0.5)
levelBonus = 1 + levelDiff × 0.05
variance   = 0.8 ~ 1.2 (随机浮动)

finalDamage = max(1, floor(baseDamage × levelBonus × variance))
```

> 最小伤害保证为 1 点，防止零伤害卡死。

### 2.2 闪避判定

```
闪避成功 = random() < dodgeChance
```

- 基础闪避值 = 敏捷/2（装备加成后重新计算）
- 部分技能提供 dodgeChance 临时加成（如醉拳）
- 轻功系统提供 **躲闪效果** 百分比加成（见第 5 章）

### 2.3 技能伤害类型

| 类型 | 机制 | 示例技能 |
|------|------|----------|
| **物伤** (damage) | 直接伤害，受攻击力/防御力影响 | 基础攻击、罗汉拳、太极剑 |
| **毒伤** (poison) | 每回合扣血，持续 N 回合 | 毒咬、毒液喷射 |
| **灼烧** (burn) | 每回合扣血，持续 N 回合 | 火龙吐息、业火 |
| **持续伤** (dot) | 每回合扣血，持续 N 回合 | 酸池 |
| **吸血** (healPercent) | 造成伤害的 X% 转化为自身 HP | 吸血、收割灵魂 |
| **吸蓝** (mpSteal) | 吸取目标 MP | 北冥神功（规划） |

### 2.4 控制效果

| 效果 | 触发方式 | 效果 |
|------|----------|------|
| **眩晕** (stun) | stunChance 概率触发 | 跳过 N 回合行动 |
| **冻结** (freeze) | freezeChance 概率触发 | 跳过 N 回合行动 |
| **恐惧** (fear) | fearChance 概率触发 | 跳过 N 回合行动 |

### 2.5 Buff/Debuff 系统

| 类型 | 方向 | 效果 |
|------|------|------|
| **buff** | 自身增益 | 修改 attack/defense/dexterity/constitution/intelligence |
| **debuff** | 敌方减益 | 修改 attack/defense/dexterity 等属性 |

效果叠加规则：同属性效果**数值累加**，独立计算持续时间。

### 2.6 特殊战斗机制

| 机制 | 说明 |
|------|------|
| **反击** (counter) | 被攻击时有 counterChance 概率反击，伤害为原伤害的 30% |
| **反伤** (reflect) | 受到攻击时对攻击者造成固定点数伤害 |
| **防御** (defend) | 防御状态，减伤处理 |
| **被动回复** | 每回合自动恢复 mpRegen/hpRegen（如冥想、洗髓经） |
| **被动加成** | 永久 buff（如易筋经 +10 体质），持续 999 回合 |

### 2.7 死亡与复活

```
死亡惩罚:
  - 经验损失: 10% (不低于 0)
  - 金币损失: 5%  (不低于 0)
  
复活:
  - HP/MP 恢复至 30%
  - 传送回新手村 (village_center)
```

## 3. 技能分类体系

### 3.1 技能层级架构（参考 PKUXKX MUD）

```
技能系统
├── 基本功夫 (Basic Skills)
│   ├── 基本内功 (force)        — 影响内力上限、恢复
│   ├── 基本轻功 (dodge)        — 影响闪避、移动
│   ├── 基本招架 (parry)        — 影响防御格挡
│   ├── 空手功夫
│   │   ├── 基本掌法 (strike)
│   │   ├── 基本拳法 (cuff)
│   │   ├── 基本手法 (hand)
│   │   ├── 基本爪法 (claw)
│   │   ├── 基本指法 (finger)
│   │   ├── 基本拳脚 (unarmed)
│   │   └── 基本腿法 (leg)
│   ├── 兵器功夫
│   │   ├── 基本匕法 (dagger)
│   │   ├── 基本刀法 (blade)
│   │   ├── 基本剑法 (sword)
│   │   ├── 基本鞭法 (whip)
│   │   ├── 基本杖法 (staff)
│   │   ├── 基本枪法 (spear)
│   │   ├── 基本锤法 (hammer)
│   │   ├── 基本戟法 (halberd)
│   │   └── 基本斧法 (axe)
│   ├── 基本暗器 (throwing)
│   └── 基本擒拿 (grappling)
│
├── 特殊功夫 (Special Skills)   — 各门派独有，需激发对应基本功夫
│   ├── 特殊内功 (如: 易筋经、九阴神功、北冥神功、葵花神功)
│   ├── 特殊轻功 (如: 凌波微步、梯云纵、金雁功)
│   ├── 特殊招架 (如: 乾坤大挪移)
│   ├── 掌法/拳法/指法/剑法/刀法/鞭法...
│   └── 经脉学 (jingmai)        — 通过通脉系统提升
│
├── 知识类技能 (Knowledge)       — 非战斗辅助技能
│   ├── 读书写字 (literate)      — 影响技能学习效率上限
│   ├── 医术 (medical-skill)
│   ├── 基本毒技 (poison)
│   ├── 鉴定之术 (identification)
│   ├── 梵文/禅宗/道学           — 宗教修炼技能
│   ├── 为官之道 (leadership)
│   └── 兵法战策 (strategy)
│
└── 公共武功 (Public Kungfu)
    ├── 金蛇系列 (金蛇剑法/金蛇游身掌/金蛇锥法)
    ├── 英雄剑法
    └── 其他可跨门派学习的武功
```

### 3.2 基本功夫 vs 特殊功夫的关系

```
特殊功夫等级 ≤ 对应基本功夫等级

示例:
  太极拳 (特殊掌法) 等级 ≤ 基本掌法 (strike) 等级
  太极剑 (特殊剑法) 等级 ≤ 基本剑法 (sword) 等级
  易筋经 (特殊内功) 等级 ≤ 基本内功 (force) 等级
```

**基本功夫 + 特殊功夫** 组合构成完整的战斗能力。基本功夫提供基础数值，特殊功夫提供招式/技能效果。

### 3.3 本项目技能类型（skills.json）

| 类型 | 说明 | 示例 |
|------|------|------|
| **attack** | 主动攻击技能 | 基础攻击、罗汉拳、太极拳 |
| **heal** | 治疗技能 | 基础治疗 |
| **buff** | 增益技能 | 狼嚎、龙啸、暗影步 |
| **debuff** | 减益技能 | 诅咒、冻结 |
| **passive** | 被动技能 | 冥想、易筋经、洗髓经 |

### 3.4 技能学习途径

| 途径 | 说明 | 对应 MUD 原版 |
|------|------|---------------|
| NPC 学习 | 消耗黄金/潜能向师傅学习 | xue 指令 |
| 秘籍阅读 | 使用 skill_book 物品概率学习 | read 指令 |
| 自行练习 | 通过特定动作提升（打树、爬树等） | practice/lian 指令 |
| 门派获得 | 加入门派后自动获得基础技能 | — |
| 任务获得 | 完成特定任务后获得秘籍或技能 | quest 奖励 |

### 3.5 勤能补拙系统（熟练度）

> 2024年新版，每个特殊武功有 10 层熟练度。

| 层数 | 描述 | 效果 |
|------|------|------|
| 0 层 | 不堪一击 | 原始基础属性（100%） |
| ... | ... | 逐层提升 |
| 10 层 | 最高 | 基础攻击属性 ×115% |

- 每日使用 `practice show / lian show` 提升
- 基础属性越高的武功，熟练度提升越慢（最快约 3 个月，最慢约 3 年）
- 500 级以上每日仅需演练一次

## 4. 等级与经验曲线

### 4.1 升级公式

```
expToNextLevel(level) = floor(expBase × level^expExponent)

其中:
  expBase     = 100    (经验基数)
  expExponent = 1.5    (经验指数)
  maxLevel    = 100    (等级上限)
```

### 4.2 各等级经验需求

| 等级 | 升级所需经验 | 累计经验 |
|------|-------------|----------|
| 1→2 | 100 | 100 |
| 2→3 | 282 | 382 |
| 3→4 | 519 | 901 |
| 4→5 | 800 | 1,701 |
| 5→6 | 1,118 | 2,819 |
| 10→11 | 3,162 | ~15,000 |
| 20→21 | 8,944 | ~85,000 |
| 30→31 | 16,431 | ~210,000 |
| 50→51 | 35,355 | ~700,000 |
| 100 | — (上限) | ~2,000,000 |

### 4.3 升级奖励

| 项目 | 数值 |
|------|------|
| 自由属性点 | +3 / 级 |
| HP 上限刷新 | calculateMaxHP() |
| MP 上限刷新 | calculateMaxMP() |
| HP/MP 回满 | current = max |

### 4.4 门派进阶

| 阶段 | 需求 | 奖励 |
|------|------|------|
| 弟子 → 执事 | 声望 ≥ 100, 等级 ≥ 10 | +5 属性点 |
| 执事 → 长老 | 声望 ≥ 500, 等级 ≥ 25 | +5 属性点 |
| 长老 → 掌门 | 声望 ≥ 2000, 等级 ≥ 50 | +5 属性点 |

### 4.5 MUD 原版经验机制（参考）

PKUXKX MUD 中经验值直接影响**技能等级上限**：

```
技能等级上限 ≈ f(经验值)

在纯 MUD 中没有"角色等级"概念，技能等级完全由经验驱动:
  - 经验越高 → 技能上限越高
  - 技能升级消耗潜能(potential)
```

本项目做了简化，采用 **等级制 + 技能书学习** 的混合模式：

- 等级决定基础属性成长
- 技能通过多种途径独立学习
- 技能等级有独立上限（requireLevel）

## 5. 轻功系统

### 5.1 轻功分级

| 级别 | 修炼消耗 | 说明 |
|------|----------|------|
| **传说** | 气血 + 内力 | 最强轻功，如辟邪剑法、凌波微步 |
| **绝学** | 气血 + 内力 | 门派顶级轻功 |
| **上乘** | 仅气血 | 中等门派轻功 |
| **普通** | 仅气血 | 基础轻功 |

### 5.2 轻功核心属性

| 属性 | 说明 |
|------|------|
| **躲闪效果** | 提高受到攻击后成功躲闪的概率 |
| **攻速效果** | 提高攻击速度 |
| **契合度** | 对应武功命中加成（如剑法契合度 10% = 用剑时 +10% 命中） |
| **暴击躲闪** | 对手暴击时的额外躲闪概率（>5% 时，一半为一招晕躲闪概率） |
| **二次躲闪** | 第一次躲闪失败后，自动激活再次躲闪的概率 |
| **修炼难度** | 影响练习消耗（如难度+5% = 消耗×1.05） |

### 5.3 限制条件

| 条件 | 效果 |
|------|------|
| **优势内功** | 激发对应优势内功时，获得**全额**躲闪效果；否则 -5% |
| **内力/精力需求** | 当前内力或精力低于需求值时，攻速效果 -10%（20M 经验后生效） |
| **属性激活** | 部分轻功需要特定 quest mark 或门派身份才能激活 buff |

## 9. 成长路径总览

### 9.1 新手成长路线

```
阶段 1: 初入江湖 (Lv 1-10)
  ├── 新手村基础任务，获得初始装备
  ├── 学习基础攻击/基础治疗
  ├── 基本轻功: 爬树(0-50)、爬墙(0-100)
  └── 读书写字: 扬州夫子(0-200)

阶段 2: 拜师学艺 (Lv 10-25)
  ├── 选择门派，学习门派专属技能
  ├── 门派任务获取声望 → 弟子→执事进阶
  ├── 基础功夫冲 100-180 级
  └── 开始收集秘籍 (射雕/tianlong 等)

阶段 3: 江湖历练 (Lv 25-50)
  ├── 执事→长老进阶
  ├── 特殊功夫收集与修炼
  ├── 轻功冲 200+ (少林跳楼/凌霄踏雪)
  ├── 开启知识技能 (梵文/禅宗/道学)
  └── 个人特征 (Trait) 选择

阶段 4: 一代宗师 (Lv 50-100)
  ├── 长老→掌门进阶
  ├── 感悟系统 (250-600 级基本功)
  ├── 勤能补拙 (熟练度 10 层)
  ├── 传说轻功获取
  └── 侠客岛深造
```

### 9.2 技能上限对照

| 阶段 | 基本功夫 | 特殊功夫 | 知识技能 | 轻功等级 |
|------|----------|----------|----------|----------|
| 新手 | 0-100 | 0-50 | 0-200 | 普通 |
| 入门 | 100-180 | 50-180 | 200-350 | 普通/上乘 |
| 熟练 | 180-250 | 180-300 | 350-500 | 上乘/绝学 |
| 精通 | 250-600 | 300-600 | 500+ | 绝学/传说 |
| 宗师 | 600+ | 600+ | 750+ | 传说 |
