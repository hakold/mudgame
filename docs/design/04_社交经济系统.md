## 1. 帮派系统 (Gang)

### 1.1 PKUXKX 帮派总览

北大侠客行的帮派系统诞生于新手导师制度的取消，是目前游戏中最重要的社交组织形态。帮派有数量上限（因地产有限），当前活跃帮派如下：

| 帮派名称 | 地址 | 
|----------|------|
| 牛肉面馆 | 临安东 |
| 广寒宫 | 西湖边 |
| 桃花学院 | 襄阳西 |
| 万梅山庄 | 泰山下 |
| 魂殿 | 苏州内 |
| 斧头帮 | 洛阳西 |
| 百花谷 | 回族小镇 |
| 断罪小学 | 铜雀台 |
| 太白楼 | 晋阳西南 |
| 黑手党 | 成都内 |

### 1.2 PKUXKX 帮派等级与人数上限

为防止一家独大，帮派采用等级制，每级有人数上限：

| 等级 | 0级(新建) | 1级 | 2级 | 3级 | 4级 | 5级 | 6级 | 7级 | 8级 | 9级 |
|------|----------|-----|-----|-----|-----|-----|-----|-----|-----|-----|
| 人数 | 10 | 20 | 25 | 35 | 40 | 50 | 55 | 60 | 65 | 70 |

升级帮派是一个缓慢的过程，升级解锁不同功能：

| 等级 | 解锁功能 |
|------|----------|
| 0级 | 基础大厅，免费食物饮水（仅本帮） |
| 2级 | 帮派狂欢(150% EXP)；库房（捐献通脉药 + 换取套装） |
| 3级 | 帮派炼丹室 |
| 4级 | 练功房可领悟；帮派锻造室 |

### 1.3 PKUXKX 建立帮派的条件

建立帮派需要两件道具及多项硬性条件：

| 条件类型 | 具体要求 |
|----------|----------|
| **建帮令** | 临安府韩世忠处获得，消耗 5万金 + 1点铜雀台积分 |
| **绿林旗** | 1000点藏经阁积分换取 |
| **存款** | 10万金以上 |
| **年龄** | 30岁以上 |
| **经验** | 100M 经验以上 |
| **现金** | 身上有 2万金存款（用于购买地产） |
| **命令** | `gcreate <帮派名>`（普通）/ `gcreate +r <帮会名>`（休闲帮派） |

**休闲帮派**：每日无积分扣除，无帮派狂欢命令，仓库不刷新装备。适合休闲玩家。

### 1.4 PKUXKX 帮派管理与命令

- **招募**: `ginvite <id>` 邀请 → 被邀请人 `gjoin <id>` 加入（邀请制，有门槛）
- **头衔**: `ggrant <某人> to <头衔>` —— 帮主、副帮主、长老、堂主、帮众
- **弹劾**: `gimpeach` 弹劾命令
- **踢人**: `gexpel <xxx>` —— 一天最多踢三个号
- **在线列表**: `glist`
- **帮派频道**: `group` 频道聊天
- **帮派狂欢**: `grevelry <分钟数>` —— 30000分启动，每周最多一次且不超过90分钟，每分钟消耗20分
- **积分查询**: `gquery -me`（个人）/ `gquery -public`（公开信息）/ `gquery -overall`（景愿级别）/ `gquery -donate`（高层查询）
- **详细信息**: `gbrief` 查看帮派详细信息
- **升级**: `upgrade`（在帮派大厅）

### 1.5 PKUXKX 帮派积分任务

每日帮派积分上限按职位：

| 职位 | 每日上限 |
|------|----------|
| 帮主 | 1500 |
| 副帮主 | 1300 |
| 长老 | 1200 |
| 堂主 | 1100 |
| 天榜 | 1500 |
| 地榜 | 1300 |

> 取最高值，不重复计算。

两种任务类型：
- **confirm zhi**: 跑腿任务，寻找资材
- **confirm qian**: 完成公孙止、萧峰、韩世忠、胡一刀、韩员外、慕容或都府刺杀任务中任意6种后，回来 `finish` 复命

额外积分来源：捐献物品给帮会库房 → 获取帮会积分；做任务几率掉落青铜器残片 → `gongxian material can pian` 换特殊帮贡。

### 1.6 PKUXKX 帮派景愿（Wish）

四大景愿（帮派次要发展方向），使用 `gongxian wish <次数>` 贡献（每天最多20次）：

| 景愿 | 1级效果 | 2级效果 | 3级效果 |
|------|---------|---------|---------|
| **降低积分扣除** | 80% | 90% | 95% |
| **增加积分任务** | 110% | 105% | 102% |
| **工匠增强** | 总管出售药材/工具 | 打造失败率-1% | 打造失败率-3% |
| **推荐入大宋** | 必然成功 | ~66% | ~33% |

贡献总愿望值达标自动升级，`gquery -overall` 查看级别。

## 2. 经济交易系统

### 2.1 PKUXKX 经济体系参考

PKUXKX 是永久免费的MUD，采用**玩家自愿捐助**维持运营，捐助不影响游戏内权限/属性/能力。游戏内经济体系包括：

- **基础货币**: 黄金（gold），通过任务/战斗/交易获得
- **门派贡献**: 门派任务获取，用于兑换门派物品/提升位阶
- **帮派积分**: 帮派任务/捐献获取，用于帮派内部兑换
- **藏经阁积分**: 副本积分，兑换高级物品
- **铜雀台积分**: 高端任务积分，用于建帮令等高级道具

### 2.2 本项目已实现的交易系统

#### 玩家间直接交易 (tradeService)

| 功能 | 实现 |
|------|------|
| 发起交易 | `trade_request` → tradeService.createTrade() |
| 放入物品 | `trade_add_item` → 可叠加放入 |
| 设置金币 | `trade_set_gold` → 双方各自设置出价 |
| 确认交易 | `trade_confirm` → 双方确认后执行交换 |
| 交易验证 | 确认时验证金币余额 + 背包物品 |
| 交易失败 | 任一方金币不足/物品变更 → 自动取消 |

**交易流程**：
```
A → trade B（发起交易请求）
  ↓
双方各自放入物品/金币
  ↓
双方确认 (trade_confirm)
  ↓
验证 → 执行交换 → 触发成就检测
```

#### 聊天系统

| 频道 | 事件 | 范围 |
|------|------|------|
| 世界频道 | `chat_world` | 全服广播 |
| 房间频道 | `chat_room` | 同一房间 |
| 私聊 | `chat_private` | 指定玩家 |
| 帮派频道 | `chat_gang` | 帮派成员 |

### 2.3 PKUXKX 互动体系参考

PKUXKX Wiki 的 hudong（互动）分区包含：
- **贡献指南**: 帮助新手、维护Wiki、论坛发帖、宣传北侠
- **捐助系统**: 自愿捐助维持服务器运营，资金明细完全公开透明
- **错别字报告**: 玩家可提交游戏中的文字错误

## 3. 拍卖行系统 (Auction)

### 3.1 本项目已实现的拍卖系统

当前项目实现了完整的拍卖行系统 (`server/src/game/auctionService.js`)：

| 功能 | 实现 |
|------|------|
| 挂单 (createListing) | 需指定数量/单价/时长 (24/48/72h) |
| 手续费 | 总价的5%（最低1金），挂单时扣除 |
| 搜索 (searchListings) | 按名称/卖家/价格区间搜索，按单价排序 |
| 购买 (buyListing) | 买方扣金→卖方收金（扣5%交易税）→物品转移 |
| 取消挂单 (cancelListing) | 退回物品，不退手续费 |
| 我的挂单 (getMyListings) | 查看所有挂单状态 |
| 过期清理 | 每5分钟自动清理过期物品 |

### 3.2 拍卖经济参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 挂单手续费 | 总价 × 5% | 挂单时扣除，不退还 |
| 交易税 | 成交价 × 5% | 从卖方所得扣 |
| 挂单时长 | 24/48/72 小时 | 到期自动过期 |
| 卖方净收入 | 成交价 - 交易税 | 自动转入 |
| 系统回收 | 手续费 + 交易税 | 控制通胀 |

### 3.3 建议扩展

- 竞拍模式（出价方式，而非固定价格购买）
- 拍卖行浏览按类别筛选项
- 拍卖历史记录/价格走势

| 天数 | 经验 | 金币 |
|------|------|------|
| 第1天 | 50 | 20 |
| 第2天 | 80 | 30 |
| 第3天 | 120 | 40 |
| 第4天 | 150 | 50 |
| 第5天 | 200 | 80 |

**连续规则**：

### 5.1 成就体系架构

```json
{
  "id": "ach_xxx",
  "name": "成就名称",
  "description": "达成条件描述",
  "category": "类别",
  "condition": { "type": "条件类型", "value": 阈值 },
```

### 5.5 成就奖励机制

```javascript
// 达成成就后
achievementService.reward(userId, achievementId)
  → 检查是否已获得（幂等）
  → 满足条件 → 创建 Achievement 记录
  → 发放 EXP + 金币 + 称号
  → 发送 achievement_earned 事件通知客户端
```

### 5.6 PKUXKX 称号参考

PKUXKX 中的称号体系更加丰富，源自多个维度（Wiki中缺乏专门的title页面，但可从其他页面推断）：

- **门派称号**: 各门派内部的道号/辈分称号（如少林"渡"字辈、"澄"字辈等）
- **天榜/地榜**: 实力排名相关称号
