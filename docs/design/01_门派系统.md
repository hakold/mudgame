## 一、门派总览

### 1.1 门派分类

侠客行MUD共设 **21个门派**（含百姓/无门派），分为以下类别：

| 分类 | 门派列表 |
|------|----------|
| **十二大门派**（有独立新手任务） | 峨嵋派、丐帮、古墓派、华山派、明教、全真派、日月神教、少林派、桃花岛、天龙寺、武当派、星宿派 |
| **纯门派**（不适合出师路线） | 华山派、星宿派、大轮寺、天龙寺 |
| **正派** | 朝廷、峨嵋派、丐帮、华山派、天龙寺、全真派、少林寺、武当派 |
| **邪派** | 白驼山、大轮寺、灵鹫宫、日月神教、神龙教、五毒教、星宿派 |
| **中立** | 古墓派、姑苏慕容、绝情谷、明教、青城派、桃花岛、天地会、雪山派 |

### 1.2 设计理念

- **无门派锁定**: 不选门派（百姓）也不影响完成常规任务、顺利成长
- **各有所长**: 每个门派都有独特定位，强者有特点、弱者并非不能玩
- **进阶驱动**: 门派威望排行影响随从等级、任务奖励、商品折扣等
- **职业匹配**: 门派与职业（青龙/白虎/朱雀/玄武）有联动加成

- 适合长期发展，无明显短板

#### 门派特色机制

1. **太极解密**: 太极神功/太极拳/太极剑>400级 → 观蛇雀相斗 → 解锁绕指柔剑(autopfm) + 三环套月翻倍 + 无需exert taiji
2. **太极真意（真传）**: 需求：铜雀台45分(大恒图)+25万任务积分(阴阳鱼) → 闭关3-6个月 → 选专精拳(仅玄武)或剑
3. **气血叠加**: 道家养生术+太极神功 → 北侠最长气血，奠定"血牛"基础
4. **道德系统**: 太极神功学习需道德支持：`道德≥2^(level/10-1)×100`，道德≥500万后不再限制

#### 进阶体系
- 谷虚道长 → 武当七侠（道德10k）→ 张三丰（悟性30+道德50k+太极神功50+新手任务）
- 新手任务：冲虚道长发布，必须完成

#### 专属任务/副本
- 武当新手任务（杀土匪 → 侠客任务）
- 武当后山探索（太子岩→雷神洞→南岩峰→潘神洞→天柱峰）
- 太极解密（观蛇雀相斗）
- 太极真意闭关（铜雀台+藏经阁材料）

#### 玩法建议
- 初始选玄武职业，天赋优先根骨
- 5W exp前做新手任务杀土匪，同时刷500W道德
- 1M后转玄武，可爬塔+韩世忠任务
- 50M过劫难
- 气血控制：保持taoism≥400即可，前期不要过高（NPC copy气血）
- 互搏必须学，影响平砍输出
