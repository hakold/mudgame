灵鹫宫

灵鹫宫可以算是逍遥派的一个分支，它是由逍遥派传人天山童姥创立的，天山童姥和无涯子、李秋水曾经同为逍遥派弟子，后来在三人之间发生了许多感情纠纷，以至于童姥和李秋水之间形同水火，双方竭尽全力想置对方于死地。童姥在感情失意后，只身来到天山灵鹫峰，招收了许多无依无靠的女人建立了一个庞大的武林门派－－灵鹫宫。后来童姥依靠师门绝技天山六阳掌和天山折梅手，在武林中掀起了滔天波涛。生死符令武林人士谈虎色变。后来，武林人士为了自身的生存，发起了一次针对童姥的行动，其时恰逢童姥练功返老还童之时，李秋水趁机出来找童姥算帐。少林和尚虚竹在此行动中，莫名其妙地得到了一身神鬼莫测的功夫，还成了逍遥派与灵鹫宫两派的掌门，而童姥、无涯子、李秋水三个同门全部去世。虚竹继任灵鹫宫掌门后，以德报怨，为众多武林人士化去生死符，令这些草莽英雄真心归附灵鹫宫，从此，灵鹫宫由原来的人所不齿的邪恶门派成为众人敬慕的武林圣地。

武功

互备：天山折梅手+天山六阳掌 bei zhemei-shou liuyang-zhang

八荒六合唯我独尊功(bahuang-gong)

八荒功作为灵鹫最具特色的本门内功，除了具有一般特殊内功的功用外，还具加强攻击力、自疗、疗人之用，更为惊奇之处是它还具有返老还童的功效。

  • 可激发为特殊内功(force)
  • 学习条件：基本内功高于20级，基本内功要高于特殊内功才能学习
  • 练习条件：不可以练习
  • 特殊功效：
    • 自疗 exert/yun heal 要求：八荒功20级，当前内力50
    • 疗人 exert/yun lifeheal 要求：八荒功40级，最大内力300，当前内力100
    • 加力 exert/yun powerup 要求：八荒功40级，当前内力100，八荒功600级以后达到1/3效果。有对有效轻功加成的效果。
    • 月华 exert/yun yuehua 要求：交给童姥七个天珠黄色残片。在系统时间为晚上时，yun yuehua可以加强灵鹫宫武学，白天不加攻击并内力全失。
    • 返老还童 huichun -r

小无相功(xiaowuxiang-gong)

  • 可激发为特殊内功(force)
  • 特殊功效：
    • 疗人 exert/yun lifeheal
    • 加力 exert/yun powerup
    • 护体 exert/yun huti
      • 功能：提高护甲。
    • 点火 exert/yun dianhuo
      • 功能：为火墙/火球做准备。星宿有用，对于灵鹫弟子没有任何用处

小无相在职业装系统下对于灵鹫来说基本无用，可以不去考虑

北冥神功(beiming-shengong)

  • 基本功效：可激发为特殊内功（force）
  • 学习条件：灵鹫宫弟子，基本内功100级，需要放弃其他内功（八荒功、小无相功、易筋经除外）
  • 练习条件：不可以练习
  • 特殊功效：
    1. 疗伤
      • 使用：exert/yun lifeheal
      • 功能：给别人疗伤，能救醒晕倒之人。
      • 要求：最大内力1000，内力500。
    2. 吸气
      • 使用：exert/yun qisuck
      • 功能：吸取临时气血。
      • 要求：北冥神功20级，空手，当前内力大于10
    3. 吸精
      • 使用：exert/yun jingsuck
      • 功能：吸取临时精神。
      • 要求：北冥神功30级，空手，当前内力大于10
    4. 吸内力
      • 使用：exert/yun neilisuck
      • 功能：吸取临时内力。
      • 要求：北冥神功40级，空手，当前内力大于20
    5. 吸永久内力
      • 使用：exert/yun maxsuck
      • 功能：吸取内力上限。
      • 要求：北冥神功80级，空手，当前内力大于20
    6. 北冥
      • 使用：exert/yun beiming
      • 功能：战斗中吸对手的内力，并且可以造成对手busy，命中不高
      • 要求：北冥神功200级，基本内功200级，最大内力1000，内力500。

月影舞步(yueying-wubu)

  • 可激发为dodge
  • 学习条件：无
  • 练习条件：气大于45，耗气
  • 特殊攻击：无
  • 轻功评价：上乘 53。
  • 基础属性：躲闪效果 4%。手法契合度 8%、暗器契合度 8%。二次闪躲 5%。
  • 特殊功能：运用『月影舞步』，降低被多人围攻时的躲闪惩罚。
  • 限制条件：优势内功 八荒六合唯我独尊功、北冥神功、小无相功。内力需求 200点。精力需求 800点。
  • 修炼难度：难度降低 5%。

天山六阳掌(liuyang-zhang)

  • 武器伤害：142%～178% 力量伤害：370%～730% 参考级别：800
  • 技能攻速：85%
  • 激发支持：暗器(throwing)、掌法(strike)、招架(parry)
  • 学习条件：空手，最大内力大于20
  • 练习条件：气大于30，当前内力大于20，耗气
  • 基本招式：落日熔金、青阳带岁除、阳歌天钧、云霞出薛帷
  • 特殊攻击：
    • 种生死符 perform liuyang-zhang.zhong
      • 功能：运内力使水结冰，给敌人造成伤害并中毒,最后会造成busy
      • 要求：身上带有酒袋，天山六阳掌激发为暗器。
      • 气势需求：发动气势：12%，消耗气势：12%，使用后气势停止增加0秒
      • 内功要求：八荒六合唯我独尊功(120)
      • 技能要求：天山六阳掌(150)、基本暗器(100)
      • 属性要求：内力(300)
      • 绝招评价：B-
    • 解生死符 perform liuyang-zhang.jie
      • 功能：解除对手的生死符之毒
      • 要求：天山六阳掌等级150， 八荒功等级120，当前内力大于200
      • 气势需求：发动气势：0%，消耗气势：0%，使用后气势停止增加0秒
      • 内功要求：八荒六合唯我独尊功(120)
      • 技能要求：天山六阳掌(150)
      • 属性要求：内力(200)
    • 连种生死符 perform liuyang-zhang.lianzhong
      • 功能：连续攻击
      • 要求：身上带有酒袋，天山六阳掌激发为暗器。
      • 气势需求：发动气势：12%，消耗气势：12%，使用后气势停止增加0秒
      • 内功要求：□八荒六合唯我独尊功 或者 小无相功(300)
      • 门派要求：仅限灵鹫宫弟子
      • 技能要求：天山六阳掌(300)、 基本暗器(300)
      • 属性要求：内力(1500)、内力修为(2000)
      • 绝招评价：C-

特别提示：六阳掌的pfm需要同时激发六阳掌为掌法和暗器，缺一不可，只激发为暗器时会提示你手上的武器无法施放种生死符。另外，激发内功为小无相功时，连种生死符cd增加为12秒。 此外，种生死符可偷袭，高几率被叫杀，作为起手，zhong+duo（没种毒就夺兵器）还是比较好用的！

成长路线

先钓鱼，后扫雪，之后门忠/慕容/破阵/送信/投名状
慕容仆人信件任务是这个时期最赚钱的任务，需要注意如果做慕容仆人信件任务，功夫要平均不要有等级差——不包括知识类，否则就变成了花样送死。有了空手夺白刃后慕容任务还是很简单的，有空多做做吧，不仅能攒钱攒经验，还能熟悉江南地图和为之后的偷学打好基础
其他的任务最好full功夫再做，尤其是送信和投名状，战斗难度很大，失败惩罚很高
个人认为破阵是这个阶段（经验<1.6M）最好的成长任务，战斗难度极低，而且只要你学会了北冥神功，破阵就不用吃药了，能省不少钱
400K经验之后可以开始押新手镖了，又多了一种成长方式。
1.5M开始进入到快速成长阶段：韩世忠吧！

长恨剑法。没有pfm，但平砍效果比天羽奇剑强些。当然初期更强的是英雄剑法。

过阵前不用学空手，读书hand和strike很难过80，还不如去读80级基本指法撑膂力，甚至不撑。

六阳掌主要是作为暗器，种生死符和连种生死符。基本不会去为别人解去生死符，所以基本掌法有就可以。

连种生死符是一个很不错的perform。伤害大，命中高，不过会百分百的叫杀，用的时候加小心。

灵鹫的一个最特殊的pfm（perform)．凤舞九天．是一个自已无busy的pfm而且有机会使对手busy,同时会有一定的叫杀机率.灵鹫的剑法里一共有两个pfm，凤舞九天，天羽分飞，凤舞九天无busy连打四下,有大约5秒的冷确时间.天羽分飞busy两秒钟，有十秒钟的冷却时间,最多一次可以连发九下。

天山折梅手：弹梅，连发三下,并有机率使对手busy。
六阳掌：生死符身上需要带有酒袋（推荐到麒麟村购买廉价酒袋，负重低价格便宜）
种生死符（化学）.打一下.可以偷袭.在不让动手的地方可以用来叫杀的。
连种生死符（物理）,连发四下。

这里要提示的是．生死符虽然攻击强．命中高．不过是百分百叫杀的，所以要用的话只有两种情况．一种是在你能百分百杀死对手．用这个pfm加快制敌速度，提高效率，要么就是自已跟对手相持不下没有胜算的时候，用这个pfm，可能一下会反败为胜.

因此说如果以上pfm组合起来使用,就是凤舞九天＋天羽分飞(busy2秒)＋弹梅.或是连种，一种16下．一种17下．从一开始发动攻击．到最后结束．再加上2秒的busy时间．就是在三秒钟的时间内可以连攻对手16或是17下.不过并不是这样的组合就能百分百制敌,根据不同的任务要有不同的用法．

凤舞九天
凤舞九天4个字，每个字一招。如果命中则根据技能等级追加数次额外攻击，如未命中，则不追加。每个字最多追加4下攻击，在每个字都命中且追加攻击全命中的情况下一共 (1+4)*4=20连。 根据经验，天羽400-410之前有一定概率追加1-3下，400-410之后必然追加3下。追加4下的条件不完全明确。天羽600、八荒600、六阳掌5xx时仍只追加3下，天羽630、八荒630、六阳掌600+时可追加4下，推测追加4下的条件为天羽、八荒、六阳掌同时达到600级。 对臂力较为敏感，据统计，217的武器无might，战斗臂力从96提升到103对劫匪的输出增幅在10%以上

天羽纷飞
出6招和9招的条件为八荒、月影、天羽等级分别达到360和540。前三招one hit伤害和平砍相当，中三招略有提高，后三招增幅很大，约有3倍。

灵鹫的功夫是以暴发力为主的，在三秒钟之内可以发出十六到十七下攻击对手，不过灵鹫没太好的busy对手的功夫，有两个perform虽然可以短暂busy对手，不过时间短的几乎可以不计算了，这是灵鹫的一个最大的弱点，可以考虑去读侠客岛上的五岳拳。

过劫难：如果前期搞到了busy的功夫，就不用说了，没有busy的话唯一能靠上的就是凤舞九天。出师时的方法就是打掉兵器，然后出一个凤舞九天看到了对手呆住了，就是你busy了对手。然后接一个连种生死符，因为大师的血很多，靠生死符打的能快点。如果对手没有呆住就先接一个弹梅，然后再种，这两下都没有使对手busy的话就得认命了。

灵鹫有三门内功：八荒、小无相、北冥。八荒与小无相为逍遥派内功，北冥设定为独立内功，兼修极易走火，因此可以看作两个不同发展方向。

八荒系

八荒与小无相同属逍遥系内功，内力通用；相对北冥系少内力，少气血。

八荒powerup效果比小无相好，有门派小招月华，能发全部perform；小无相yun shield，task很有用；一般主修八荒。

北冥系

北冥系高内力（前提是高先天根骨），高气血，不用打坐内力。无门派小招，缺perform纷飞，perform连种。

基本上，可以当作自己已经出师没有六脉的天龙了。

小无相系

纯门派完全不如八荒，但星宿/大轮出师过来路线真的存在。 http://pkuxkx.net/forum/thread-43181-1-1.html

任务解析

先要说的是如果练灵鹫的时候没有学习任何busy对手的功夫的话．那么初期就不要考虚萧峰跟胡一刀．硬要作的话你会死的很惨的,

慕容

不用多说．见面后面接连发．能打多少就就打多少下．基本上就是这样．也有碰到变态叛徒，从见面一直busy你．直到最后你死在他的手上．这样的npc我碰到过．也死在了他的手上．不过玩了这么久只碰到过一个.

task

这个任务对灵鹫来说是一个很不错的任务．有一个小技法．就是作这个任务的时候．把内功jifa到小无相功上．然后身上什么也不穿．只穿一个布衣．手里拿一把好剑．推荐玄铁．打npc的时候yun powerup;yun shield．见面后先把对手的兵器用折梅手的perform hand.duo打掉．然再用连击．打掉兵器后就要以连种生死符为主要进攻手段。可以快的杀死对手．这样作的原因是因为yun shield后虽然你自身什么也没有穿．不过shield加的防护足够你自已使用．而tasknpc是copy你的当前状态的．对手防护很低．也只穿一件布衣的防护,你打中对手后他很容易受伤．使最大气血下降,而你不会受伤．所以你只要吸气就可以了,对手的最大气血掉下去后就很容易打死.

护镖

纷飞冷却时间已经从一分钟降低到了十秒，应该是可以随便用了
对灵鹫来说.前期护镖还是不错的．不过这里要有一个pfm顺序，碰到第一个匪的时候．要用先用天羽分飞去打,有百分之九十五以上的机率是一下把对手匪打跑，再走第二步的时候用凤舞九天接弹梅．再走第三步的时候还是用天羽分飞．这样用是因为天羽分飞有一定的cooltime(冷确时间).

萧峰和胡一刀

萧峰跟胡一刀放在一起说．如果没有busy对手功夫的话就不要去作了．刚开始的时候如果学到busy对手功夫的话．就是见面后busy对手．然后用夺打掉对手兵器．萧峰任务以折梅手弹梅为主.手里要拿一个轻剑.比如七星剑.或是游侠剑.打中杀手后杀手不易受伤.这样可以节省金创药跟给对手疗伤的时间.折梅手里的perform折梅是一下一下的打（明教的不反弹),胡一刀也是一样．busy对手后．记得打掉兵器．左右手的全要打掉，因为安全第一.heihei

护送

现在应该是已经没有人做护送了
重点说一下护送任务:这个任务现在是北侠里公认的成长最快的一个任务．一轮下来exp基本都在十万上下．前提是你要有一个好的队友．目前我只跟古墓的跟天龙的人一起组队．感觉都很爽．天龙的人能给大师疗伤．古墓有玄铁剑．打中对手后．对手有短暂的busy.而自已在发动攻击时也要用凤舞九天或是弹梅这两个pfm放在最前面.因为这两个pfm也有使对手有短暂busy的机会，特别注意的是一定要把内功jifa到小无相功上．因为小无相功的yun lifehal XXX可以用小无相功的短busy换掉别人的长busy.在护送的时候．大师或是队员被黑衣杀手busy的时候 是很多的。
