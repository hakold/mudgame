## Phase 5：客户端UI补全

> 目标：让服务端已有功能在客户端完整呈现，消除体验断层

### 5.1 地面掉落面板
- **文件**: `client/src/views/Game.vue`, `client/src/stores/game.js`
- **北侠参考**: 北侠中kill后loot尸体获取物品，地面物品是核心交互
- **实现**:
  - 房间描述下方新增"地面物品"区域
  - 监听 `room_drops` / `room_drops_updated` 事件
  - 每个物品显示：名称、数量、拾取按钮
  - 点击拾取发送 `pickup_item` 事件
  - 空地面不显示该区域

### 5.2 死亡复活界面
- **文件**: `client/src/views/Game.vue`
- **北侠参考**: 北侠有假死/真死/鬼门关/门派疗伤/残生契等多层死亡机制
- **实现**:
  - 检测玩家HP=0时显示死亡遮罩
  - 显示"你已倒下..."提示 + 复活按钮
  - 点击复活发送 `revive` 事件
  - 复活后显示惩罚信息（经验/金币/耐久损失）

### 5.3 属性点分配
- **文件**: `client/src/views/Game.vue`
- **北侠参考**: 北侠levelup系统，升级获得天赋点，可分配到四维(str/con/dex/int)
- **实现**:
  - 角色面板显示 `freePoints` 可用点数
  - 每个基础属性旁显示 "+" 按钮（freePoints>0时可用）
  - 点击发送 `allocate_points` 事件
  - 分配后实时更新属性值和HP/MP

### 5.4 门派信息面板
- **文件**: `client/src/views/Game.vue`
- **北侠参考**: 北侠门派体系：拜师→学艺→出师→门忠→首席，门派任务/门派比武
- **实现**:
  - 角色面板显示：门派名、等级(弟子/执事/长老/掌门)、声望、贡献
  - 门派进阶按钮（满足条件时高亮）
  - 门派任务按钮（捐献金币换声望）
  - 退出门派按钮（确认对话框）

### 5.5 装备耐久度
- **文件**: `client/src/views/Game.vue`
- **北侠参考**: 北侠装备有损耗，铁匠修复
- **实现**:
  - 背包中装备显示耐久条（当前/最大）
  - 耐久低于30%变红色警告
  - 铁匠房间显示"修复装备"按钮
  - 修复按钮：单件修复 / 全部修复
  - 修复费用显示

### 5.6 战斗日志查看
- **文件**: `client/src/views/Game.vue`
- **北侠参考**: 北侠combat_brief战斗简报
- **实现**:
  - 主界面新增"战斗记录"标签页
  - 列表显示：日期、怪物名、结果(胜/败)、获得经验
  - 点击查看战斗详情（回合数据、技能使用、伤害统计）
  - 分页加载（limit/offset）

### 5.7 技能等级与经验
- **文件**: `client/src/views/Game.vue`
- **北侠参考**: 北侠skills命令显示技能级别、熟练度、本命技能
- **实现**:
  - 技能面板显示：等级、经验条、下一级所需经验
  - 战斗中技能升级时弹出通知
  - 技能类型图标区分（攻击/治疗/被动/增益）

### 5.8 休息按钮改进
- **文件**: `client/src/views/Game.vue`
- **北侠参考**: 北侠exert heal恢复内力，客栈休息恢复
- **实现**:
  - 休息按钮始终可见（不再只在客栈显示）
  - 客栈/酒馆：显示"客栈休息（完全恢复）"
  - 野外：显示"就地休息（部分恢复）"
  - 自然恢复提示（每60秒HP/MP微增）

## Phase 6：社交与互动

> 目标：建立玩家间互动基础，不含组队和好友系统

### 6.1 聊天系统增强
- **文件**: `server/src/socket/index.js`, `client/src/views/Game.vue`
- **北侠参考**: 北侠有chat频道(江湖/门派/帮派/私聊)，emote系统，say命令带语气
- **实现**:
  - 频道分类：世界(chat)、区域(room)、私聊(whisper)
  - 聊天消息带频道标识和颜色区分
  - 私聊：`whisper <玩家名> <内容>`
  - 消息气泡样式优化（区分系统/玩家/自己）

### 6.2 玩家间交易
- **文件**: 新增 `server/src/game/tradeService.js`, `server/src/socket/index.js`
- **北侠参考**: 北侠give命令直接给物品/金币，当铺sell/buy，储物柜
- **实现**:
  - 交易请求：`trade_request <玩家名>` → 对方收到确认
  - 交易界面：双方放入物品/金币 → 双方确认 → 交换
  - 交易安全：双方确认才生效，任一取消则作废
  - 距离限制：同一房间才能交易

### 6.3 PVP竞技场
- **文件**: `server/src/game/battleService.js`, `server/src/socket/index.js`
- **北侠参考**: 北侠比武大会、kill命令、武林公敌、江湖追杀令
- **实现**:
  - 竞技场房间（city_arena）发起PVP挑战
  - 挑战请求 → 对方接受 → 进入战斗
  - 复用现有战斗系统，对手替换为玩家数据
  - PVP无死亡惩罚，失败方HP=1
  - 排行榜：胜场/胜率/等级

### 6.4 玩家在线列表
- **文件**: `server/src/socket/index.js`, `client/src/views/Game.vue`
- **北侠参考**: 北侠who命令查看在线玩家，who -l详细信息
- **实现**:
  - 服务端维护在线玩家列表
  - 客户端显示在线玩家（名称、等级、门派、位置）
  - 同房间玩家高亮显示
  - `who` 命令查看全服在线

## Phase 7：内容深度扩展

> 目标：增加长期可玩性，丰富游戏内容

### 7.1 门派专属技能树
- **文件**: `config/json/skills.json`, `server/src/socket/index.js`
- **北侠参考**: 北侠每个门派有独特武功体系（少林72绝技、武当太极系列、古墓玉女心经等），门派等级限制技能学习
- **实现**:
  - 每个门派填充10-15个技能，按rankRequired分级：
    - **弟子级**(5-8个): 基础门派武功
    - **执事级**(2-3个): 中级绝学，需门派等级deacon
    - **长老级**(1-2个): 高级绝学，需门派等级elder
    - **掌门级**(0-1个): 镇派绝学，需门派等级leader
  - 每个技能有独特效果（新buff/debuff/被动/连击等）

### 7.2 门派任务链
- **文件**: `config/json/quests.json`, `server/src/game/questProgressService.js`
- **北侠参考**: 北侠门忠任务（mz命令），门派日常（捐献/护送/刺杀），门派比武
- **实现**:
  - 每个门派3-5个专属任务：
    - **护送任务**: 护送NPC从A到B（简化：到达指定房间）
    - **收集任务**: 收集门派所需材料
    - **除害任务**: 击杀指定怪物
    - **修炼任务**: 使用指定技能N次
  - 门派任务奖励：声望+经验+门派专属物品

### 7.3 副本系统
- **文件**: 新增 `server/src/game/instanceService.js`, `server/src/socket/index.js`
- **北侠参考**: 北侠剑心居（独立副本空间）、暗杀副本、铜雀台
- **实现**:
  - 副本入口：特定NPC或房间触发
  - 副本类型：
    - **试炼副本**: 单人挑战，逐层推进（5-10层）
    - **秘境副本**: 探索收集，限时30分钟
    - **BOSS副本**: 单个强力BOSS，稀有掉落
  - 副本机制：
    - 独立房间实例，不与其他玩家共享
    - 进入消耗：体力/门票物品
    - 每日次数限制（3次）
    - 通关奖励：经验+金币+稀有物品

### 7.4 帮派系统
- **文件**: 新增 `server/src/models/Gang.js`, `server/src/game/gangService.js`
- **北侠参考**: 北侠帮派(gang)：创建/加入/帮战/领地/glist/gretire/帮派福利
- **实现**:
  - 创建帮派：需5级+1000金，设置名称和宗旨
  - 帮派等级：1-5级，升级需帮派资金和成员数
  - 帮派成员：帮主/副帮主/长老/帮众，权限不同
  - 帮派功能：
    - 帮派聊天频道
    - 帮派仓库（共享物品）
    - 帮派任务（集体目标）
    - 帮战（预约制PVP）
  - 帮派福利：帮派等级提供经验/金币加成

### 7.5 生活技能
- **文件**: 新增 `server/src/game/craftService.js`, `config/json/recipes.json`
- **北侠参考**: 北侠庖丁解牛(烹饪)、锻造、炼药、认穴(针灸)、演奏(乐理)
- **实现**:
  - 四大生活技能：
    - **采集**: 在野外房间采集材料（草药/矿石/木材）
    - **锻造**: 用矿石+图纸打造武器/防具
    - **炼药**: 用草药炼制药品（比商店更强）
    - **烹饪**: 用食材制作食物（临时buff）
  - 生活技能等级：1-100级，使用升级
  - 配方系统：recipes.json定义输入/输出/等级要求
  - 采集点：特定房间有采集资源，刷新时间5-15分钟

### 7.6 成就系统
- **文件**: 新增 `server/src/models/Achievement.js`, `server/src/game/achievementService.js`
- **北侠参考**: 北侠goal系统（泡遍天下、千锤百炼等），称号系统
- **实现**:
  - 成就分类：
    - **战斗**: 首杀/百杀/击杀BOSS/PVP首胜
    - **探索**: 到达所有地图/发现隐藏房间
    - **收集**: 收集全套装备/材料
    - **社交**: 交易次数/帮派成员
    - **门派**: 门派满级/学会所有门派技能
  - 成就奖励：称号（显示在角色名旁）+ 属性加成
  - 成就进度追踪（如"击杀野狼 15/50"）

## Phase 8：经济与平衡

> 目标：建立可持续的经济循环，调优数值平衡

### 8.1 拍卖行
- **文件**: 新增 `server/src/models/Auction.js`, `server/src/game/auctionService.js`
- **北侠参考**: 北侠当铺系统（sell/buy/list/category），范蠡查询全服商品
- **实现**:
  - 玩家挂单：设定物品、数量、单价、时长(24h/48h/72h)
  - 买家搜索：按名称/类型/等级筛选
  - 购买流程：选择→确认→扣款→物品入背包
  - 挂单费用：5%手续费
  - 过期处理：未售出物品退回邮箱

### 8.2 经济平衡
- **文件**: `config/json/` 各配置文件
- **北侠参考**: 北侠金币产出（任务/卖物）与消耗（学技能/修装备/买房）的平衡
- **实现**:
  - 金币产出审计：任务奖励/怪物掉落/卖物品
  - 金币消耗审计：学技能/修装备/买物品/帮派/交易税
  - 调整原则：
    - 1-10级：新手期，金币充裕，快速成长
    - 10-30级：成长期，收支平衡
    - 30-50级：挑战期，需要精打细算
    - 50+级：高端期，稀有物品消耗大量金币
  - 通胀控制：拍卖行手续费/修理费/高级消耗品

### 8.3 数值平衡
- **文件**: `config/json/` 各配置文件, `server/src/game/battleService.js`
- **北侠参考**: 北侠战斗公式从exp系数改为技能级别系数，弱点攻击分类
- **实现**:
  - 战斗公式审查：确保属性/技能/装备都有合理影响
  - 怪物数值曲线：HP/攻击/防御按等级平滑增长
  - 技能伤害平衡：同等级技能DPS相近，特色在效果不在数值
  - 装备数值曲线：每10级一个档次，属性提升明显
  - 经验曲线：确保升级时间合理（1-10约1小时，10-30约5小时）

### 8.4 每日活跃
- **文件**: `server/src/game/dailyService.js`, `server/src/socket/index.js`
- **北侠参考**: 北侠日活系统（签到/每日任务/限时活动/双倍经验）
- **实现**:
  - 每日签到：连续签到奖励递增（7天一轮）
  - 每日任务：3个随机任务（击杀/采集/对话），完成给额外奖励
  - 活跃度系统：完成活动获得活跃点，满30/60/100给奖励
  - 限时活动：周末双倍经验/节日特殊任务

### 8.5 天气与时间
- **文件**: 新增 `server/src/game/worldService.js`, `server/src/socket/index.js`
- **北侠参考**: 北侠昼夜系统（夜间大别山危险/暗坊NPC夜间出现），天气影响（彩虹/温泉）
- **实现**:
  - 游戏内时间：1现实小时=6游戏时辰，12时辰=1天
  - 昼夜交替：
    - 白天(辰时-酉时)：正常
    - 夜晚(戌时-卯时)：部分NPC不出现，野外怪物更强
  - 天气系统：
    - 晴/阴/雨/雪/雾 随机切换
    - 雨天：采集效率+50%
    - 雾天：怪物闪避+10%
    - 雪天：移动消耗MP
  - 客户端显示当前时辰和天气
